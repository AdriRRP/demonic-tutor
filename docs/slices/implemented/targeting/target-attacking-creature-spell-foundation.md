# Target Attacking Creature Spell Foundation

## Summary

Introduce an explicit legal-target rule for spells that may target exactly one attacking creature.

## Scope

- add `AttackingCreature` as a supported creature-target rule
- allow supported spell cards to carry that rule on their card face
- reject clearly invalid target kinds such as players

## Out Of Scope

- richer combat-relative targeting such as blocking, controlled attacking, or multiple targets
