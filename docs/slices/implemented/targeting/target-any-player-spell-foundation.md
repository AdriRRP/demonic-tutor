# Target Any Player Spell Foundation

## Status

Implemented

## Scope

- the base targeted-spell subset includes an explicit unrestricted player-target rule
- a supported spell with that rule may target any existing player in the two-player game
- the unrestricted player-target rule is distinct from `opponent player`

## Out Of Scope

- multiplayer targeting semantics
- multiple player targets
- broader `any target` semantics
