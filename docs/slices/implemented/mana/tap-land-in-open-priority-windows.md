# Tap Land In Open Priority Windows

## Status

Implemented

## Scope

- the active player may tap a land for generic mana while holding priority in a currently supported empty-stack priority window
- the current exercised windows are `Upkeep` and `BeginningOfCombat`
- the land still must be on the battlefield, untapped, and controlled by the acting player

## Out Of Scope

- non-active mana production during responses
- mana abilities that explicitly bypass the stack model
- colored mana
