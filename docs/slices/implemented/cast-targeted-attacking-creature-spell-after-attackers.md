# Cast Targeted Attacking Creature Spell After Attackers

## Summary

Allow a supported targeted instant with the `AttackingCreature` rule to be cast in the post-attackers priority window against the declared attacker.

## Scope

- use the post-attackers priority window that already exists
- require the chosen target to be the attacking creature
- preserve the explicit target on the stack object

## Out Of Scope

- broader combat-relative target families
