# Slice Implemented - Support Cast From Graveyard On Explicit Card Profile

## Outcome

One explicit supported card profile can now be cast from its controller's own graveyard at normal timing and cost.

## Supported Behavior

- casting permission profile carries an explicit `CastFromOwnGraveyard` rule
- the spell leaves graveyard, uses the normal stack corridor, and resolves to its normal destination
- the current supported profile is explicit and narrow, not a generic flashback or escape system
