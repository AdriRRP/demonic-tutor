# Controlled Attacking Creature Spell

## Summary

Allow a supported targeted instant to require a creature that is both attacking and controlled by the acting player or controller.

## Scope

- add an explicit closed legal-target rule for `controlled attacking creature`
- prove the rule in the current post-attackers active-priority window
- reject combat creatures that do not satisfy both ownership and attacking status
- cover both lethal and nonlethal damage consequences for the supported spell profile

## Out Of Scope

- broader actor-relative attacking restrictions beyond the current closed target set
