# Controlled Blocking Creature Spell

## Summary

Allow a supported targeted instant to require a creature that both blocks and is controlled by the acting player or controller.

## Scope

- add an explicit closed legal-target rule for `controlled blocking creature`
- prove the rule in the current post-blockers response window for the defending player
- reject combat creatures that do not satisfy both ownership and blocking status

## Out Of Scope

- richer combinations beyond the supported closed combat-relative target set
