# Release 0.8.0 - Wave 1 - Counters And Combat Triggers

Goal:

- finish the two highest-return battlefield snowballing patterns that were intentionally pushed out of `0.7.0`

Slice count:

- `2`

Slices:

- `support-distribute-plus-one-plus-one-counters`
  - support placing counters across one or more chosen creatures in a bounded targeting subset
- `support-attack-and-combat-damage-triggers`
  - support first explicit attack triggers and combat-damage-to-player triggers

Why this wave has high return:

- it closes the biggest remaining gap between the current board engine and a believable curated limited environment without reopening the `0.7.0` release scope
