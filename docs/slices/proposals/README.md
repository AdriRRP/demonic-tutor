# Slice Proposals

This directory contains the live proposal backlog.

## Playable Horizon

The current planning horizon to reach an honestly playable limited subset is:

- `3` releases
- `9` waves
- `36` slices

The planning gates are:

- after `0.7.0`, the public gameplay contract should be stable enough to start UI work without inventing rules in the frontend
- after `0.8.0`, the engine should support enough common limited-card patterns to build a curated set skeleton
- after `0.9.0`, the project should be able to support real best-of-one games with a deliberately constrained limited set

See:

- [playable-horizon.md](/Users/adrianramos/Repos/demonictutor/docs/slices/proposals/playable-horizon.md)
