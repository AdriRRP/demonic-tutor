# Slice Proposals

This directory contains the live proposal backlog.

## Playable Horizon

The current planning horizon to reach an honestly playable limited subset is:

- `2` releases
- `10` waves
- `36` slices

The planning gates are:

- after `0.7.0`, the public gameplay contract should be stable enough to start UI work without inventing rules in the frontend
- after `0.8.0`, the project should be able to support real best-of-one games with a deliberately constrained curated limited set

## Release Status

`0.7.0` is now feature-complete on `main`.

That means:

- the public gameplay snapshot and legal-action surface are stable enough to support the first UI without inventing domain semantics in the frontend
- the remaining backlog in this directory now belongs to `0.8.0`, not to unfinished `0.7.0` scope
- proposal cleanup should not keep already implemented `0.7.0` waves looking like future work
