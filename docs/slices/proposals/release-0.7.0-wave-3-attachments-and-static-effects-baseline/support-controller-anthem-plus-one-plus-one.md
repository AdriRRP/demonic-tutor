# Slice Proposal - Support Controller Anthem Plus One Plus One

## Goal

Support the first non-attachment static battlefield effect:

- creatures controlled by one player get `+1/+1` while the anthem permanent remains on the battlefield

## Scope

In:

- one explicit controller-scoped anthem profile
- battlefield stat projection that includes the anthem bonus
- deterministic stacking with existing creature counters and temporary bonuses

Out:

- color-restricted anthems
- keyword-granting anthems
- full continuous effect layers

## Why This Matters

This is the cheapest first step from spell-only value into board-shaped permanent strategy.
