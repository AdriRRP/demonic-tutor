# Release 0.7.0 - Wave 3 - Attachments And Static Effects Baseline

Goal:

- add the first battlefield-resident effects that keep mattering after resolution

Slice count:

- `4`

Slices:

- `support-creature-aura-casting-and-attachment`
  - support aura spells that target a creature and enter attached to it
- `support-stat-boost-auras`
  - support explicit attached `+N/+N` modifications in the bounded subset
- `support-pacifism-style-auras`
  - support simple attached attack/block restrictions from an aura
- `support-controller-anthem-plus-one-plus-one`
  - support a first static battlefield effect that boosts creatures controlled by one player

Why this wave has high return:

- auras and static boosts are among the cheapest ways to make a limited set feel like real Magic rather than only stack tricks
