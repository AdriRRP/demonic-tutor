# Release Notes - v0.6.0

## Summary

`v0.6.0` is the release where DemonicTutor stops feeling like only a well-architected engine core and starts feeling like a much more usable Magic subset.

The main theme of this release is:

- broader stack interaction
- first real triggered-ability gameplay
- much more usable activated abilities
- richer combat and keyword coverage
- meaningful graveyard and token play patterns

---

## Highlights

- Stack interaction now includes counterspells, bounce, artifact/enchantment removal, and targeted discard.
- Supported triggered abilities now use the stack for `ETB`, `dies`, `upkeep`, and `end step`.
- Activated abilities now cover tap abilities, mana-costed abilities, sacrifice costs, and the first supported planeswalker loyalty corridor.
- Combat now supports multiple blockers, ordered damage assignment, `Deathtouch`, and `Double strike`.
- Tokens and `+1/+1` counters now give the battlefield much more texture.
- Graveyard gameplay now includes return-to-hand, honest reanimation scope, mill, and explicit cast-from-own-graveyard profiles.
- Keyword support expanded with `Menace`, `Lifelink`, `Hexproof`, and `Indestructible`.

---

## Why It Matters

`v0.5.0` primarily hardened the runtime.

`v0.6.0` spends that architectural capital on gameplay:

- more meaningful interaction on the stack
- more permanents that are worth activating and sequencing
- more combat states that produce recognizably Magic-like outcomes
- more deck patterns involving graveyard value, tokens, and counters

This does not mean “full Magic rules support”.

It does mean the current subset is now substantially more playable, expressive, and useful for real deck-playtesting experiments.

## Upgrade Note

This release materially expands the supported gameplay surface, but it still remains a constrained subset.

The most important practical change is that the engine can now exercise many more realistic card patterns without lying about unsupported rules:

- more honest targeting and graveyard semantics
- broader timing and trigger coverage
- more useful combat outcomes
- more varied permanent gameplay
